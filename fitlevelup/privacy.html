Política de Privacidad de FitLevelUp

Última actualización: 4 de julio de 2026

FitLevelUp respeta tu privacidad. Esta Política de Privacidad explica qué datos recogemos, para qué los usamos y qué derechos tienes cuando utilizas la aplicación FitLevelUp.

1. Responsable

Responsable de la aplicación: FitLevelUp

Contacto de privacidad: soporte.fitlevelup.app@proton.me

Sitio web o página de soporte: https://charyster.github.io/fitlevelup/

2. Datos que podemos recoger

  • Datos de cuenta: correo electrónico, identificador de usuario y datos necesarios para iniciar sesión.
  • Datos de perfil: alias, género, provincia, edad, altura, peso, nivel, experiencia, avatar y, si decides subirla, foto de perfil.
  • Datos de entrenamiento: ejercicios seleccionados, día de entrenamiento, series, repeticiones, descansos, pesos y progreso guardado.
  • Datos de comunidad: mensajes publicados, provincia o ámbito de comunidad, alias, nivel, foto de perfil si la tienes activada, quedadas creadas, asistentes y mensajes dentro de quedadas.
  • Datos técnicos y de uso: información necesaria para mantener la sesión, sincronizar datos y guardar preferencias en el dispositivo.

No solicitamos datos de salud médica ni información clínica. Los datos de entrenamiento y perfil se usan para personalizar la experiencia deportiva dentro de la app.

3. Cómo usamos tus datos

  • Crear y gestionar tu cuenta.
  • Guardar tu perfil y tus preferencias.
  • Mostrar tus entrenamientos, rutinas, progreso y niveles.
  • Permitir funciones de comunidad, mensajes y quedadas.
  • Sincronizar datos entre dispositivos cuando inicias sesión.
  • Mejorar la estabilidad, seguridad y funcionamiento de la app.
  • Atender solicitudes de soporte o eliminación de cuenta.

4. Base legal

Tratamos tus datos principalmente para prestar el servicio solicitado por ti al crear y usar una cuenta en FitLevelUp. En algunos casos, también podemos tratar datos por interés legítimo, por ejemplo para seguridad, prevención de abuso y mantenimiento técnico de la aplicación.

Cuando subes una foto de perfil o publicas mensajes en la comunidad, lo haces de forma voluntaria.

5. Comunidad y contenido visible para otros usuarios

Si usas las funciones de comunidad, otros usuarios autenticados pueden ver ciertos datos asociados a tus publicaciones, como tu alias, nivel, provincia o ámbito de comunidad, mensajes, quedadas creadas, asistentes y foto de perfil si la tienes activada.

Evita publicar información sensible, privada o de terceros en chats, mensajes o quedadas.

6. Almacenamiento y proveedores

FitLevelUp utiliza Supabase para autenticación, base de datos y sincronización de información. Supabase puede tratar datos necesarios para prestar estos servicios bajo sus propias condiciones y medidas de seguridad.

La app también puede guardar datos localmente en tu dispositivo, por ejemplo preferencias, progreso o copias locales de tu estado de entrenamiento, para que la experiencia sea más rápida y estable.

7. Conservación de datos

Conservamos tus datos mientras mantengas tu cuenta o mientras sean necesarios para prestar el servicio.

Algunos datos de comunidad pueden tener limpieza automática o límites de retención para mantener la app ordenada. Si eliminas tu cuenta, se eliminarán los datos asociados a tu usuario conforme a la configuración de la base de datos y a las obligaciones legales aplicables.

8. Eliminación de cuenta

Puedes solicitar o realizar la eliminación de tu cuenta desde la propia app si la opción está disponible. Al eliminar la cuenta, se cerrará la sesión y se eliminarán los datos asociados a tu usuario en el sistema principal.

También puedes contactar con nosotros en soporte.fitlevelup.app@proton.me para solicitar acceso, corrección o eliminación de tus datos.

9. Seguridad

Aplicamos medidas razonables para proteger tus datos, incluyendo autenticación de usuario, políticas de acceso en la base de datos y separación de datos por cuenta. Aun así, ningún sistema conectado a internet puede garantizar seguridad absoluta.

10. Menores

FitLevelUp no está dirigida a menores de 13 años. Si detectamos que se ha creado una cuenta incumpliendo este requisito, podremos eliminarla.

11. Transferencias internacionales

Al usar proveedores tecnológicos como Supabase, tus datos pueden tratarse en servidores ubicados fuera de tu país. En esos casos, se aplicarán las garantías y condiciones ofrecidas por dichos proveedores.

12. Cambios en esta política

Podemos actualizar esta Política de Privacidad cuando cambie la app, sus funcionalidades o los requisitos legales. Publicaremos la fecha de última actualización al inicio del documento.

13. Contacto

Para cualquier consulta sobre privacidad o protección de datos, puedes escribir a:

soporte.fitlevelup.app@proton.me
